# FitCare Client Project

## Programming Languages and Technology Used
- Java
- XML
- Firebase Realtime Database
- Firebase Authentication
- Firebase Cloud Storage

## Features Involved 
- Push Notifications
- Text To Speech
- Voice Command Navigation
- CRUD Operation
- BMI Calculation
- Exercise and Stretcbes Videos
